\# Trello Realtime Backend



This is the backend for the Trello Realtime App, built using Node.js, Express, and WebSocket.  

It handles tasks, boards, and real-time updates via Trello webhooks.



\## Features

\- REST API for tasks and boards (`/api/tasks`, `/api/boards`)

\- Webhook endpoint for Trello realtime events (`/webhook`)

\- WebSocket setup for broadcasting realtime updates to frontend

\- Error handling middleware

\- CORS enabled and JSON parsing
